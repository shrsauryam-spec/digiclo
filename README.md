<p align="center">A Digital Clock with Date, Day, Temperature, and Alarm System using NodeMCU ESP8266</p>

## Why?
I do not usually use my phone, so the only way for me to know the time is with a clock or a watch. And I do not wear a watch. At the same time, I do not have a watch in my room, so I created this Digital Clock to help me find out the current time. Another thing is I have a habit of forgetting the date and the day, which isn't really great. So, this clock can help me stay up-to-date. 

## Key Features:
1. Wemos D1 Mini ESP8266 as NodeMCU (WiFi Connectivity)
2. Date, Time, Day, and Temperature Display on a 1.8-inch TFT Display
3. Alarm System using a Buzzer
4. Pomodoro Timer with a Potentiometer
5. Battery Charging with TP4056 Charging Module with Protection

## Electrical Components Required:
1. Wemos D1 Mini ESP8266
2. TFT LCD Display - 1.8" inch
3. LiPo Rechargeable Battery 2000 mAh 3.7V
4. TP4056 Charging Module with Protection
